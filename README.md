# Rust-sandbox
For learning Rust lang.

## Folders
### basic
* basic types
* basic collections
* casting
* control flow
* functions
* trait

### middle
* box pointer
* closure
* iterator
* generic programming
* trait enhanced

### high
* macro
* error_handle
* concurrency
* tests

### projects
Real projects, like games, network tools, etc.

#### snake

* A simple game, simulates GREEDY SNAKE.
* Rust 2018 Edition.
* With piston_window <0.107.0>, draw_2d<x,y,z>

#### ip_sniffer
* A port scanner
* usage: ip_sniffer [-h|j] [number of threads] ipaddr
